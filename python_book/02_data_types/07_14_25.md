# sets

sets use unique members so there are no duplicates.
```python
>>> s4 = {0, 3, 2, 5, 5, 3}
>>> print(s4)
{0, 2, 3, 5}
```

### set types
* class set - unordered collection of objects, you can do mathematical operations with them.
* class frozenset - immutable sets
