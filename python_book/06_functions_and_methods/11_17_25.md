# interning, identity
two objects can have the same values and different ids `id()`

### `dir()`

* on its own returns all identifiers available in local scope
* `dir(variable)` returns a list of the objects methods and attributes.
    * returns a list of methods and attributes including "dunder" or double underscore names `__name__` `__reversed__`.
    * the output can be formatted with `sorted()` and `for in` expressions.

### `help()`
Use help to identify object methods and attributes

### creating functions

```py
def newFunction(): # assign function to 'newFunction'
    print("my new function")
```

`newFunction()` is a function call, when python hits the `()` of the function it jumps to where the function is defined in the program.
