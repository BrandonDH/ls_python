# functions and methods 
print multiple arguments
```
print('bob', 'greg', 'sandwich')
```

`print()` returns `None`. `print("greg")` returns `None`, and prints out "greg".

## Built-in Functions
* `min()`, `max()` - collections values
* strings, sequences, mappings, sets are iterable
* `ord()` - unicode point of a character: 
```py
print(ord('a')) # 97
```
* `char()` - inverse of `ord()`

Built in introspection tools: `help(func)`, `dir(obj)`

## truthy and falsey
* truthy and falsey describe values. they are not the same as true or false. they are more general.

* falsey: 
    * `False`, `None`
    * numeric `0` values (int, float, complex(?))
    * Strings
    * empty collections: `[]`, `()`, `{}`, `set()`, `frozenset()`, `range()`
    * custom data types

* truthy:
    * everything else

## `any` and `all`

useful built-in functions, operate on iterable collections.
